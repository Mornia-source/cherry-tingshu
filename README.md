# 樱桃听书

用本地 AI 角色声线朗读电子书的桌面 Web 应用。

樱桃听书把电子书（txt / epub / pdf）解析成逐句文本，调用本地语音引擎（GPT-SoVITS / IndexTTS）合成指定角色音色的语音，支持逐句高亮朗读、整章预生成、多角色声线切换、个性化阅读和后台管理。整套程序在本机运行，语音不经过第三方服务。

本仓库**不包含语音引擎和语音模型本体**（体积大，且涉及第三方版权）。需要你按下文自行下载，并在网页里填写路径，程序会自动接入。

## 功能

- 电子书解析：txt / epub / pdf 自动分章、逐句切分，自动提取封面。
- 两种语音引擎：
  - GPT-SoVITS：需要训练角色模型，速度快，适合日常实时听书。
  - IndexTTS2：免训练，提供一段参考音频即可克隆音色，适合临时加角色或预生成。
- 逐句朗读高亮，倍速调节。
- 整章预生成：提前合成整章并缓存，之后秒开；支持断点续传和随时停止。
- 多角色声线：网页内添加、重命名、切换。
- 主题换肤，LOGO 随主题变色。
- 账号系统：注册登录、头像、阅读进度、个性化设置。
- 后台管理：引擎开关、模型训练、声线管理、预生成音频库导入导出、账号总览。

## 目录结构

```
eBookSVC/
├── app/                后端（FastAPI）
├── web/                前端（原生 HTML/CSS/JS）
├── ebook/              电子书（不上传）
├── model/              语音模型（不上传，自行放置）
├── data/               账号库、封面、预生成缓存（不上传）
├── config.json         个人配置（不上传，首次运行自动生成）
├── config.example.json 配置模板
├── 启动听书.bat        一键启动（GPT-SoVITS + Web）
├── 停止听书.bat        一键停止所有服务
├── start_api.bat       单独启动 GPT-SoVITS
└── start_indextts.bat  单独启动 IndexTTS
```

## 环境要求

- Windows 10 / 11
- Python 3.10 及以上
- NVIDIA 显卡，建议显存不低于 6GB（8GB 可用，但建议一次只跑一个引擎）

## 安装

### 1. 获取项目与依赖

```bash
git clone <本仓库地址>
cd eBookSVC
pip install -r requirements.txt
```

### 2. 下载语音引擎（按需，二选一或都装）

两个引擎不随本项目分发，请自行下载，放到任意位置后在网页「设置 - 语音引擎」里填写根目录，程序会自动接入。

GPT-SoVITS（作者：花儿不哭 / RVC-Boss）

- 官方仓库：https://github.com/RVC-Boss/GPT-SoVITS
- 推荐下载作者发布的 Windows 整合包（自带便携 Python，免配环境）。
- 解压后确认目录里有 `api_v2.py` 和 `runtime/python.exe`。
- 在「设置 - 语音引擎 - GPT-SoVITS」填入该目录，或直接放到本项目根目录（会自动探测）。

IndexTTS2（免训练克隆）

- 官方仓库：https://github.com/index-tts/index-tts
- 按其说明用 uv 安装依赖、下载 checkpoints 模型。
- 在「设置 - 语音引擎 - IndexTTS」填入其根目录（含 `.venv` 与 `checkpoints`）。

### 3. 准备语音模型（角色音色）

角色模型不在本仓库内。GPT-SoVITS 既可自己训练，也可使用社区分享的现成模型。常用资源（请遵守各自的使用授权）：

- 花儿不哭（GPT-SoVITS 作者）的教程与示例；
- AI Hobby / AI 爱好者论坛：社区分享的 GPT-SoVITS 角色模型；
- 白菜工厂：游戏 / 动漫角色语音素材与模型分享；
- B 站搜索「GPT-SoVITS 角色模型」「游戏角色 语音模型」等关键词。

下载到的模型（`*.ckpt`、`*.pth`）和参考音频放进 `model/<角色名>/`，再到网页里添加该声线。IndexTTS 角色更简单：在「添加 IndexTTS 声线」里填名称、一段参考音频路径、以及这段音频对应的文字即可，无需训练。

## 运行

- 启动：双击 `启动听书.bat`（启动 GPT-SoVITS + Web），浏览器访问 http://127.0.0.1:8000 。
- 按需启动 IndexTTS：双击 `start_indextts.bat`，或在网页设置页用开关启动。
- 停止：双击 `停止听书.bat`。

显存提示：GPT-SoVITS 与 IndexTTS 同时运行会争抢显存。8GB 显卡建议一次只开一个；要用 IndexTTS 预生成时，先关掉 GPT-SoVITS 再启动 IndexTTS。

## 使用

1. 注册并登录。默认管理员账号 `Mornia`，密码 `mornia`，首次登录后请立即修改密码；普通用户可自行注册。
2. 在「书架」导入电子书。
3. 打开书，右上角选择角色声线与倍速。
4. 点播放即逐句朗读；或点「预生成」把整章先合成好再收听。
5. 设置页可管理引擎开关、声线、预生成音频库。

## 预生成音频集（按书名识别）

在「设置 - 本地预训练音频管理」里可以导出整库或单章为 zip，也可以导入他人分享的预生成包。导入时会读取每个音频集里记录的书名，自动匹配到本机的同名书籍并归位，因此换电脑或换路径后也能按书名识别使用，无需重新生成。

## 常见问题

- 引擎启动不了或一直加载：多半是上次的进程没关干净占用了端口。先双击 `停止听书.bat`，再启动。
- 网页提示「引擎未启动」：全局状态会跟随当前所选声线对应的引擎。用 IndexTTS 声线时只需启动 IndexTTS。
- 合成很慢：低显存设备建议用「预生成」挂着跑，生成完即可秒播。
- 首次运行没有 config.json：程序会自动从 `config.example.json` 生成，无需手动创建。
